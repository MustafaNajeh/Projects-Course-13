## 📌 نظرة عامة

هذا المستودع يحتوي على مشاريع **كورس 13** المُنفَّذة بلغة **C++**، وهي مشاريع تعليمية تتدرج من الأساسيات إلى المفاهيم المتقدمة في البرمجة كائنية التوجه وهياكل البيانات.

---

## 🗂️ المشاريع

### 📁 المشروع الأول — Project 1
> **المتطلبات والأساسيات**

اللبنة الأولى في رحلة تعلم C++، يُرسي قواعد البرمجة ويُحدد متطلبات البرنامج بوضوح.

**✨ أبرز المميزات:**
- 📋 تحليل وتحديد متطلبات البرنامج
- 🔤 أساسيات بنية برنامج C++
- ⌨️ التعامل مع المدخلات والمخرجات
- 🔧 المتغيرات وأنواع البيانات الأساسية

---

### 📁 المشروع الثاني — Project 2
> **العمليات والمنطق البرمجي**

يتعمق في العمليات الحسابية والمنطقية مع بناء قرارات برمجية متقنة.

**✨ أبرز المميزات:**
- ➕ العمليات الحسابية والمنطقية والمقارنة
- 🔀 هياكل التحكم `if / else / switch`
- 🔢 التعامل مع أنواع البيانات المتعددة
- 💡 بناء منطق القرار في البرنامج

---

### 📁 المشروع الثالث — Project 3
> **الحلقات التكرارية والخوارزميات**

تطبيق شامل للحلقات التكرارية مع خوارزميات بحث وحل مسائل عملية.

**✨ أبرز المميزات:**
- 🔄 حلقات `for / while / do-while`
- 🔁 التكرار المتداخل والمعقد
- 🔍 خوارزميات البحث الأساسية
- ⏹️ منطق التوقف `break` والاستمرار `continue`

---

### 📁 المشروع الرابع — Project 4
> **الدوال والإجراءات**

يُجسّد مبادئ تنظيم الكود وإعادة استخدامه من خلال الدوال والإجراءات.

**✨ أبرز المميزات:**
- 🧩 تعريف الدوال واستدعاؤها
- 📨 تمرير المعاملات (by value / by reference)
- 🔙 القيم المُرجَعة ونطاق المتغيرات
- ♻️ إعادة استخدام الكود وتنظيمه

---

### 📁 المشروع الخامس — Project 5
> **المصفوفات والمؤشرات**

يستعرض هياكل البيانات الأساسية مع التعامل المباشر مع الذاكرة.

**✨ أبرز المميزات:**
- 📊 المصفوفات أحادية ومتعددة الأبعاد
- 👉 المؤشرات `Pointers` والذاكرة الديناميكية
- 🔃 خوارزميات الفرز (Bubble Sort / Selection Sort)
- 🔎 خوارزميات البحث الثنائي

---

### 📁 المشروع السادس — Project 6
> **البرمجة كائنية التوجه — OOP**

بوابة دخول عالم OOP من خلال بناء كلاسات وكائنات حقيقية.

**✨ أبرز المميزات:**
- 🏛️ تعريف الكلاسات `Classes` والكائنات `Objects`
- 🔒 التغليف `Encapsulation` وإخفاء البيانات
- 🏗️ الدوال البانية `Constructor` والهادمة `Destructor`
- 🎯 الدوال الأعضاء وتحديد الصلاحيات (`public / private`)

---

### 📁 المشروع السابع — Project 7
> **الوراثة والتعدد الشكلي**

تطبيق متقدم يبني نظاماً هرمياً للكلاسات باستخدام الوراثة والتعدد الشكلي.

**✨ أبرز المميزات:**
- 🧬 الوراثة البسيطة والمتعددة `Inheritance`
- 🎭 التعدد الشكلي `Polymorphism`
- 🔮 الدوال الافتراضية `Virtual Functions`
- 🎨 التجريد `Abstraction` وتصميم الكلاسات الهرمية

---

### 📁 المشروع الثامن — Project 8
> **المشروع الختامي المتكامل**

تاج المشاريع الذي يجمع كل المفاهيم المكتسبة في نظام متكامل واحتراف.

**✨ أبرز المميزات:**
- 💾 القراءة والكتابة في الملفات `File Handling`
- ⚠️ معالجة الاستثناءات `Exception Handling`
- 🔧 إدارة الأخطاء وضمان جودة البرنامج
- 🏆 تكامل جميع مفاهيم الكورس في مشروع واحد

---

| المتطلب | الإصدار |
|---------|---------|
| مترجم C++ | GCC 9+ أو MSVC 2019+ |
| معيار اللغة | C++17 أو أحدث |
| بيئة التطوير | VS Code / Visual Studio / CLion |

## 🚀 طريقة التشغيل

```bash

cd Projects-Course-13/Project\ 1/

# تجميع البرنامج

# تشغيل البرنامج
./output
```
